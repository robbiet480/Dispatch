Build 22 (version 1.0)

NEW: THE REPORTER HOME SCREEN
- The home screen now honors the original: full-bleed stacked charts with labels inside the blocks, the question as a small uppercase heading, a left-aligned "+ Filter Visualizations" row, plain page dots, and an AWAKE pill toggle in a reserved bottom strip. Charts can never hide behind the dots again — structurally.

NEW: MEDIA + SMARTER CONNECTION
- Reports can capture what you're listening to: Apple Music automatically (with the Media sensor on), Spotify via Settings > Sensors > Connect. If neither is playing but something is audible, that's noted too.
- The connection sensor now knows Wi-Fi, Wired, 5G, LTE, 3G, 2G, and Satellite — not just "Cellular."

IMPROVED
- Sensors settings: each permission shows its real status inline — Granted, a Request button, or Denied (tap to open Settings). Request All lives at the bottom and disappears when there's nothing left to ask.
- Fixed black text on teal in the question editor, choices editor, prompt groups editor, and more (light and dark).
- Returning from Spotify authorization no longer strands you at the Face ID lock — the app re-prompts and finishes the connection after unlock.
- First launch is quieter and more correct: no notification-scheduling churn before you've granted permission, and iCloud backups write reliably on first run.

WHAT TO TEST
1. The new home screen: swipe your questions, tap the AWAKE pill, check every theme in light + dark.
2. Settings > Sensors: check your permission statuses match reality; try a Request button.
3. Connect Spotify (Settings > Sensors > Media), play a song, file a report — with the app lock enabled.
4. File a report on 5G or satellite if you get the chance — check the connection row.
