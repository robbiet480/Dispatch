Build 11 — Focus Filters

NEW: Dispatch now appears under Settings > Focus > [any mode] > Focus Filters.
Add it to a Focus and you can:
- Give the Focus a name that gets captured with every report ("Work" instead of just "Focus: on").
- Choose which Prompt Groups keep asking questions during that Focus — leave the group list untouched to keep everything, or select specific groups to silence the rest.
- Optionally pause the main prompt schedule too.
When the Focus ends, your full schedule comes right back. Snoozed reminders always fire — an explicit snooze outranks the filter.

WHAT TO TEST
1. Add the Dispatch filter to Do Not Disturb with a name + one group selected. Toggle DND on: pending prompts should narrow to that group. Toggle off: full schedule returns (check Settings > Notifications for the filter status row while active).
2. Switch directly between two Focus modes that both have Dispatch filters — the second one should win.
3. Configure a filter with ONLY a name (do not touch the group list) — prompts should be unaffected, and reports filed during that Focus should show the name.
